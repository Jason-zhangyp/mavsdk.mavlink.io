enum Result

Possible results returned for param requests.

Value Description
UNKNOWN Unknown error.
SUCCESS Request succeeded.
TIMEOUT Request timed out.
CONNECTION_ERROR Connection error.
WRONG_TYPE Error.
PARAM_NAME_TOO_LONG Parameter name too long (> 16).
